# SHD GmbH Landing Page

This repository contains the mobile-first design for the **SHD GmbH Landing Page**.

## Overview 🚀
The project is built specifically with mobile optimizations in mind, ensuring a seamless experience across all handheld devices. It utilizes pure HTML and modern aesthetics to showcase the brand's layout effectively.

## Features ✨
- **Mobile Optimized**: Custom tailored strictly for smaller resolutions.
- **Modern UI**: Smooth components and structured layouts.
- **Fast Delivery**: Pure semantic HTML for optimal page load speed.

## Quick Start ⚙️
To run this project locally, simply clone the repository and open `index.html` in your favorite web browser.

```bash
git clone https://github.com/anhnguyen102517-hash/shd-gmbh-landing-page.git
```
